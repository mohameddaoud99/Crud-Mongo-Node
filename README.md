# CRUD-mongodb-express-ejs
**CRUD (Crear, Leer, Actualizar y Borrar)** mongodb, express y ejs

### Use and installation

Install modules: `npm install`
Start: `node app.js`

RUN in: **127.0.0.1:3000**
